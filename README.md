# HAI Mastery

Study webapp for the **History of AI** course final exam (VU Amsterdam, May 2026).

**Live:** [hai-mastery.vercel.app](https://hai-mastery.vercel.app)

## What's inside

- **Overview** — exam format, high-probability topics, and study workflow
- **Topic Mastery** — all A/B/C topics with key people, dates, connections, and exam tips. Mark topics as mastered to track progress.
- **Questions + Answers** — past exam questions with model answers, filterable by type and exam session
- **Connections** — concept-to-question map, bridge facts, and question pattern toolkit
- **People & Dates** — reference table of key figures + a chronological timeline
- **Cheat Sheet** — dense single-page review for exam morning
- **Practice** — random question drill with draft textarea and word count
- **Answer Templates** — essay skeletons for standard, spectrum, and process questions

## Stack

Single self-contained HTML file — no build step, no dependencies, no backend. Deployed via Vercel.
